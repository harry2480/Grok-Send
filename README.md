# Grok-Send

このブラウザ拡張機能は、Grok (x.ai) および X.com (Twitter) 上の Grok において、**Enter** キー単体での意図しないメッセージ送信を防止し、**Ctrl + Enter** (Mac の場合は **Cmd + Enter** も可) で送信するように変更します。

## 主な機能

- **誤送信の防止:** Enter キーのみを押した場合は、送信されずに改行が挿入されます。
- **ショートカット送信:** `Ctrl + Enter` (Windows/Linux) または `Cmd + Enter` (Mac) でメッセージを送信できます。
- **マルチドメイン対応:** `grok.com` および `x.com/i/grok` の両方に対応しています。
- **IME（日本語入力）対応:** 日本語入力中の変換確定（Enter）を妨げない設計です。
- **確実なイベント制御:** ブラウザのキャプチャリングフェーズでイベントを捕捉するため、Grok 側のリスナーを確実にバイパスします。

## インストール方法（開発者向け）

1. このリポジトリをクローンまたはダウンロードします。
2. フォルダ内で `npm install` を実行し、依存関係をインストールします。
3. `npm run build` を実行して拡張機能をビルドします。
4. Chrome ブラウザを開き、`chrome://extensions/`（拡張機能管理画面）にアクセスします。
5. 右上の「**デベロッパー モード**」をオンにします。
6. 「**パッケージ化されていない拡張機能を読み込む**」をクリックし、ビルドされた `dist` フォルダを選択してください。

## ビルドスクリプト

- `npm run dev`: 開発モード（ホットリロード有効）で起動します。
- `npm run build`: 本番用にビルドします。
